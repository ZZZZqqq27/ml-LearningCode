# 吴恩达机器学习课程 p26-p50 总结

本项目是吴恩达机器学习课程 p26-p50 内容的总结和示例代码，包含逻辑回归、过拟合问题与解决方案、神经网络与深度学习入门等内容。

## 项目结构

```
p26-p50/
├── 1_逻辑回归.ipynb                # 逻辑回归模型实现
├── 2_过拟合问题与解决方案.ipynb      # 过拟合识别与正则化技术
├── 3_神经网络与深度学习入门.ipynb    # 神经网络基础与实现
└── README.md                        # 项目说明文档
```

## 内容说明

1. **逻辑回归**
   - 二分类数据生成
   - Sigmoid激活函数
   - 逻辑回归成本函数
   - 梯度下降算法
   - 决策边界可视化
   - 模型预测与评估

2. **过拟合问题与解决方案**
   - 过拟合示例数据生成
   - 不同阶数多项式拟合对比
   - 正则化线性回归（岭回归）
   - 正则化逻辑回归
   - 不同正则化参数效果对比
   - 过拟合解决方法总结

3. **神经网络与深度学习入门**
   - 神经网络基本结构
   - 常用激活函数（Sigmoid、ReLU、tanh）
   - 前向传播实现
   - 使用NumPy实现简单神经网络
   - TensorFlow基础介绍
   - 深度学习应用领域

## 环境要求

- Python 3.7+
- NumPy
- Matplotlib
- TensorFlow (可选，用于TensorFlow示例)

## 安装依赖

```bash
pip install numpy matplotlib tensorflow
```

## 使用方法

1. 克隆或下载本项目到本地
2. 确保已安装所需依赖
3. 使用 Jupyter Notebook 或 Jupyter Lab 打开相应的 .ipynb 文件
4. 按顺序运行每个代码单元格，查看结果和可视化

## 学习建议

1. 从 `1_逻辑回归.ipynb` 开始，逐步学习后续内容
2. 理解逻辑回归与线性回归的区别
3. 掌握过拟合问题的识别和解决方法
4. 理解神经网络的基本结构和前向传播
5. 尝试修改代码中的参数，观察结果变化
6. 结合吴恩达课程的理论知识，加深对深度学习的理解

- 每个示例都有数据生成、模型训练、评估和可视化部分
