# 吴恩达机器学习课程 p51-p70 总结

本项目是吴恩达机器学习课程 p51-p70 内容的总结和示例代码，包含矩阵乘法、激活函数、Softmax、多输出分类和高级优化等内容。

## 项目结构

```
p51-p70/
├── 1_矩阵乘法.ipynb                  # 矩阵乘法的原理和实现
├── 2_激活函数.ipynb                   # 激活函数的选择和实现
├── 3_Softmax激活函数.ipynb            # Softmax函数及其应用
├── 4_多输出分类与高级优化.ipynb        # 多类/多标签分类和Adam优化
└── README.md                          # 项目说明文档
```

## 内容说明

1. **矩阵乘法**
   - 矩阵乘法的基本原理
   - 手动实现与NumPy实现对比
   - 性能分析
   - TensorFlow中的实现
   - 在神经网络中的应用

2. **激活函数**
   - 激活函数的重要性
   - Sigmoid函数及其局限性
   - 替代方案：tanh、ReLU、Leaky ReLU、ELU
   - 激活函数的选择指南
   - 多类别分类问题

3. **Softmax激活函数**
   - Softmax函数的原理
   - 数值稳定的实现
   - 与神经网络的集成
   - 交叉熵损失函数
   - 梯度计算

4. **多输出分类与高级优化**
   - 多类分类与多标签分类的区别
   - 多类分类的实现
   - 多标签分类的实现
   - Adam优化算法
   - 额外的层类型
   - 计算图概念
   - 深层神经网络实现

## 环境要求

- Python 3.7+
- NumPy
- Matplotlib
- TensorFlow (可选，用于TensorFlow示例)
- scikit-learn (可选，用于数据生成)

## 安装依赖

```bash
pip install numpy matplotlib tensorflow scikit-learn
```

## 使用方法

1. 克隆或下载本项目到本地
2. 确保已安装所需依赖
3. 使用 Jupyter Notebook 或 Jupyter Lab 打开相应的 .ipynb 文件
4. 按顺序运行每个代码单元格，查看结果和可视化

## 学习建议

1. 从 `1_矩阵乘法.ipynb` 开始，逐步学习后续内容
2. 理解矩阵乘法在神经网络中的核心作用
3. 掌握不同激活函数的特性和适用场景
4. 理解Softmax函数在多类别分类中的应用
5. 区分多类分类和多标签分类的实现差异
6. 掌握Adam等高级优化算法的原理
7. 尝试构建和训练更大的神经网络

## 注意事项

- 所有代码都包含详细注释，便于理解
- 每个示例都有数据生成、模型训练、评估和可视化部分
