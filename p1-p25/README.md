# 吴恩达机器学习课程 p1-p25 总结

本项目是吴恩达机器学习课程 p1-p25 内容的总结和示例代码，包含线性回归、多特征、向量化、梯度下降、特征缩放、特征工程、多项式回归等内容，以及使用 scikit-learn 库的示例。

## 项目结构

```
p1-p25/
├── 1_线性回归.ipynb            # 单变量线性回归示例
├── 2_多特征线性回归.ipynb       # 多特征线性回归示例
├── 3_向量化实现.ipynb           # 向量化实现与性能对比
├── 4_特征缩放.ipynb            # 特征缩放（标准化和归一化）
├── 5_特征工程.ipynb            # 特征工程（交互特征、多项式特征）
├── 6_多项式回归.ipynb           # 多项式回归与过拟合分析
├── 7_scikit-learn示例.ipynb     # 使用scikit-learn库的示例
└── README.md                    # 项目说明文档
```

## 内容说明

1. **线性回归**
   - 单变量线性回归模型实现
   - 梯度下降算法
   - 代价函数计算
   - 模型可视化

2. **多特征线性回归**
   - 多特征数据生成
   - 模型训练与评估
   - 特征重要性分析

3. **向量化实现**
   - 向量化与非向量化实现对比
   - 性能测试
   - 向量化梯度下降

4. **特征缩放**
   - 标准化（Standardization）
   - 归一化（Min-Max Scaling）
   - 特征缩放对梯度下降的影响

5. **特征工程**
   - 交互特征创建
   - 多项式特征创建
   - 不同特征工程方法的性能对比

6. **多项式回归**
   - 非线性数据拟合
   - 不同阶数多项式对比
   - 过拟合分析
   - 模型选择

7. **scikit-learn示例**
   - 使用scikit-learn实现线性回归
   - 使用scikit-learn实现多项式回归
   - 模型评估与可视化

## 环境要求

- Python 3.7+
- NumPy
- Matplotlib
- scikit-learn

## 安装依赖

```bash
pip install numpy matplotlib scikit-learn
```

## 使用方法

1. 克隆或下载本项目到本地
2. 确保已安装所需依赖
3. 使用 Jupyter Notebook 或 Jupyter Lab 打开相应的 .ipynb 文件
4. 按顺序运行每个代码单元格，查看结果和可视化

## 学习建议

1. 从 `1_线性回归.ipynb` 开始，逐步学习后续内容
2. 理解每个算法的原理和实现细节
3. 尝试修改代码中的参数，观察结果变化
4. 结合吴恩达课程的理论知识，加深对机器学习算法的理解

## 注意事项

- 所有代码都包含详细注释，便于理解
- 每个示例都有数据生成、模型训练、评估和可视化部分
